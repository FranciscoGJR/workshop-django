# Workshop Django
## Roteiro
---

### Instalar ferramentas
**Python**
Verificar se já veio instalado no GitHub CodeSpaces:
```
python --version
```

**Django**
Instalar:
```
python -m pip install Django
```

Verificar se foi instalado no GitHub CodeSpaces:
```
python -m django --version
```

**curl**
Verificar se já veio instalado no GitHub CodeSpaces:
```
curl --version
```
